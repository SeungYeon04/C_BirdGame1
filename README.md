
#### 어려웠던 점  
다른 스크립트에서 가져온 머니 변수  
다른 씬에서 100 가져온 후 변동 일어나도  
씬 이동 한 번 하면 0이던 머니변수도 100으로 돌아옴  
아마 그 가져오기만 했지 그 변수는 변화가 안 일어난 것 같음  
프리팹을 사용해서 그런가 싶어서 다른 거 구현한 후  
더 공부 후 수정하려고 계획중
